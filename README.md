## Technologies
- Languages: JavaScript (ES6+), HTML (EJS templates), CSS  
- Runtime: Node.js  
- Framework: Express.js  
- Templating: EJS  
- Database: MongoDB (via Mongoose)  
- File uploads: Multer, Cloudinary  
- Session management: express-session
